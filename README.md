# Black Parrot Tavern - One Page Website

A complete static one-page website for the **Black Parrot Tavern** restaurant-bar concept.

## How to run

Open `index.html` in a browser.

## Structure

- `index.html` - main page
- `assets/css/style.css` - styles
- `assets/js/script.js` - mobile menu, menu filter, lightbox
- `assets/images/brand` - logo and website background
- `assets/images/menu` - food and drink images
- `assets/images/interior` - interior images

## GitHub Pages

Upload the folder contents to a repository and enable GitHub Pages for the `main` or `master` branch.
